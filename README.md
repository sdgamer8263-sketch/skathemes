## вҡЎ Install (One Command)

```bash
cd /var/www/pterodactyl
blueprint -install skatheme
```

**Or download + install manually:**

```bash
wget https://github.com/sdgamer8263-sketch/skathemes/raw/main/skatheme.blueprint
blueprint -install skatheme.blueprint
```

## рҹ“Ұ What is SKA Theme?

SKA Theme is a **Blueprint extension** for Pterodactyl Panel вҖ” same format as Nebula, Drift, and other Blueprint themes.

- вң… Install with one `blueprint -install` command
- вң… Admin panel settings page built-in
- вң… Deep-space neon purple & cyan palette
- вң… Glassmorphic sidebar, tooltips, context menus
- вң… Animated server cards & status orbs
- вң… Keyboard shortcuts, floating windows, file mode
- вң… Hardcoded `Copyright В© SKA` footer
- вң… Configurable second footer line via admin panel

## вҡҷпёҸ Admin Settings

After install, go to:

**Admin Panel вҶ’ Extensions вҶ’ SKA Theme**

From there you can configure:
- Color palette (dashboard, sidebar, auth page, status colors)
- Sidebar layout (icon-only or full labels)
- Server list style (cards or list)
- Animations (fadeup, zoomin, fadein, disabled)
- Alert/announcement bar
- Quick-link weblinks bar
- Keyboard shortcuts toggle
- Border radius
- Custom logo (sidebar + login)
- Background images
- Transparency levels
- Configurable second footer line (`Design by SKA`)

## рҹ”’ Footer

```
Copyright В©пёҸ SKA          вҶҗ HARDCODED вҖ” cannot be changed
Design by SKA             вҶҗ CONFIGURABLE вҖ” edit in Admin вҶ’ SKA Theme
```

The first line is permanently embedded in `client/wrapper.blade.php` and cannot be removed or changed through any admin interface.

The second line is controlled via the admin settings page or by setting `SKA_DESIGN_BY` in `.env`.
